СУБД для сайта Postgres, его нужно скачать(пароль для субд установите как admin)

Пароль для СУБД admin, если у вас другой пароль, то меняем его в \bride\server\.env в поле DB_PASSWORD


В Windows заходим в консоль WIN + R => в поле ввода пишем cmd => вводим команду: cd bride => вводим команду: cd server
=> вводим команду(запускаем сервер): npm run dev

В Windows заходим в консоль WIN + R => в поле ввода пишем: cmd => вводим команду: cd bride => вводим команду: cd client
=> вводим команду(запускаем клиент): npm start

На сайте реализованы каталог(добавление и просмотр типов и товаров) и корзина (добавление и просмотр товаров)

Чтобы добавить в корзину, создаём пользователя и в каталоге у выбранного товара нажимаем кнопку "В избранное"

Чтобы добавить тип или товар в каталог, заходим в postgres(СУБД), в табице с пользователями(users) у выбранного
пользователя меняем поле role с USER на ADMIN и обязательно сохраняем изменённые данные
Теперь в каталоге админ сможет изменять данные о товаре
